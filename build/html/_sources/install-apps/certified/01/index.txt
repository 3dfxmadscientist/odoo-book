Customer Relationship Management - CRM Application
==================================================

.. note::
	Before you proceed for the installation you should know the difference between :ref:`OpenERP Apps vs Module <get-start-module-apps>`. If you are going to install modules that is available locally, make sure your are on latest version of that modules. Its better to work with the OpenERP Apps store that always deliver latest in terms of ``versions`` along with the ``bug fix`` and ``improvements``.

Install
-------
To install Customer Relation Module - CRM in OpenERP, Goto Settings  → Apps  → and click on Install button of CRM module.

.. image:: images/crm-module.png

CRM Module

It may ask you for the username and password for OpenERP.com account on successfully login it will download best version for CRM module from apps store. Once you install CRM module you are ready to working with the leads by :ref:`creating manually <leads-manual>` or :ref:`create automated <leads-automatic>` leads through website.
