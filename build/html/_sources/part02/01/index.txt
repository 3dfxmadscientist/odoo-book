Install CRM Module in OpenERP
=============================

To install Customer Relation Module - CRM in OpenERP, Goto Settings -> Apps -> and click on Installed button of CRM module.

.. image:: images/crm-module.png

CRM Module

It may ask you for the username and password for OpenERP.com account on successfully login it will download best version for CRM module from apps store. Once you install CRM module you are ready to working with the leads by :ref:`creating manually <leads-manual>` or :ref:`create automated <leads-automatic>` leads through website.
