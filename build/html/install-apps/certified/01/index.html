Customer Relationship Management - CRM Application¶

Note

Before you proceed for the installation you should know the difference between OpenERP Apps vs Module. If you are going to install modules that is available locally, make sure your are on latest version of that modules. Its better to work with the OpenERP Apps store that always deliver latest in terms of versions along with the bug fix and improvements.

Install¶

To install Customer Relation Module - CRM in OpenERP, Goto Settings → Apps → and click on Install button of CRM module.

CRM Module

It may ask you for the username and password for OpenERP.com account on successfully login it will download best version for CRM module from apps store. Once you install CRM module you are ready to working with the leads by creating manually or create automated leads through website.
